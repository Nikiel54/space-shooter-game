#  🚀 SPACE DEFENDERS

Play as a lone space fighter defending their base from a neverending onslaught of asteroids.
Aim for a highscore, try to beat my personal best of 644 points!

## 🎮 **[PLAY THE GAME](https://nikiel54.github.io/space-shooter-game/)**

*Click the link above to play in your browser!*

## How to Play

- Use WASD to move
- Press Left Mouseclick to fire lasers
- Avoid and destroy incoming debris
- Collect powerups and play for a highscore!

## Features

- Smooth pygame graphics
- Scaling difficulty system
- Score system

## Local Development

To run locally:

```bash
git clone https://nikiel54.github.io/space-shooter-game/
cd space-shooter-game
pip install pygame
python main.py
```

## Built With

- Python & Pygame
- Deployed with pygbag to GitHub Pages

## Controls

| Key | Action |
|-----|---------|
| WASD | Move |
| Left CLick | Shoot|
| ESC | Exit |
